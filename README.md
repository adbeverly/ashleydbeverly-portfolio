# ashleydbeverly.dev

Personal portfolio and projects — built in public.

## About
Terminal-style portfolio site built by Ashley D. Beverly.
Self-taught developer with 10+ years of experience at Rocket Mortgage
spanning sales, underwriting, and enterprise software development.

## Features
- Interactive terminal interface
- Ask My Resume — AI chat powered by Anthropic API
- Visitor weather log — real-time location and weather via Firestore
- Admin easter egg — try typing `admin` at the prompt

## Built With
- HTML, CSS, JavaScript
- Firebase Firestore + Authentication
- Anthropic API
- OpenWeatherMap API
- Cloudflare Workers
- GitHub Pages
